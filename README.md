# Plantar Huautli

¡Bienvenido a Plantar Huautli! Este proyecto tiene como objetivo fomentar la plantación de árboles y concienciar sobre la importancia de cuidar el medio ambiente.

## Descripción del Proyecto

Plantar Huautli es una plataforma interactiva que permite a los usuarios plantar árboles en ubicaciones específicas y realizar un seguimiento de su crecimiento. Además, los usuarios pueden ganar puntos por sus contribuciones y participar en misiones para obtener recompensas adicionales.

## Funcionalidades Principales

- **Mapa Interactivo:** Explora y selecciona ubicaciones para plantar árboles en un mapa interactivo.
- **Notificaciones:** Recibe notificaciones sobre el estado de tus árboles y actualizaciones del juego.
- **Acciones Personalizadas:** Realiza acciones como tomar fotos y plantar semillas para ganar puntos.
- **Estadísticas y Puntos:** Consulta tus estadísticas, incluidos los puntos acumulados y el progreso de tus árboles.

## Tecnologías Utilizadas

- React.js para la interfaz de usuario.
- Material-UI para el diseño y la interfaz.
- Google Maps API para la visualización del mapa.
-web3
